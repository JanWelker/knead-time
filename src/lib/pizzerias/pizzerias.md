# 50 Top Pizza pizzerias

Pizzerias from the [50 Top Pizza](https://www.50toppizza.it/) guide whose
dough recipes are documented publicly (interview, cookbook, official article,
chef's own video). Each row encodes the published recipe as a Knead Time
share URL — open one and tweak the bake time to your day.

The **Rankings** column is a comma-separated list of `YEAR-LIST:RANK` entries.
`LIST` is `it` for 2018–2021 (when 50 Top Pizza was an Italy-only guide) and
`w` for 2022–2025 (the standalone World ranking). Add a row only when you can
cite a primary source for the dough numbers.

The **Timing** column captures the proving durations the source recipe
specifies, as `step-kind:Nh` or `step-kind:N-Mh` (or `Nm` for minutes).
Recognised step kinds: `preferment-mix`, `bulk-room`, `bulk-cold`,
`final-proof`. Schedule rows whose computed duration falls outside the
source range get the original value tagged inline.

The **Notes** column flags anything Knead Time's encoding can't represent
faithfully — a specialty flour blend, an extra ingredient the model doesn't
support, or "approximation" when the source doesn't pin gram-precise numbers.
Keep it short; leave it empty when the recipe maps cleanly.
